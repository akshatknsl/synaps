DeleteAlarms¶

Description¶

It deletes all alarm that is specified in action parameter. If at least one AlarmName does not exist in the database, this action will not affect to any alarm.

Parameters¶

Name Description Mandatory
AlarmNames.member.N

List of alarm name to delete

Data type: String list

Limitation: 0 ~ 100

 No

see also Common Query Parameters

Errors¶

Error Description HTTP Status Code
ResourceNotFound Requested alarms are not found 404
