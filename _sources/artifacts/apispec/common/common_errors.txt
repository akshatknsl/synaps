.. _common_errors:

Common Errors
=============

It describes common errors for all API actions.

.. list-table:: 
   :widths: 20 50 10
   :header-rows: 1
   
   * - error
     - description
     - HTTP Status Code
   * - InvalidSignature
     - Signature of request is invalid
     - 400
   * - InvalidRequest
     - Request is invalid
     - 400
   * - InternalFailure
     - Request is failed by unknown internal error. 
     - 500
