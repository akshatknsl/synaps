EnableAlarmActions
==================

Description
----
Activate action of specific Alarm.

Parameters
-------------

.. list-table:: 
   :widths: 20 50 10
   :header-rows: 1

   * - Name
     - Description
     - Mandatory
   * - AlarmNames.member.N
     - List of Alarm name to activate action.

       Data type: String list

       Length limitation: 0 ~ 100 items
     - Yes

see also :ref:`common_query_parameters`
